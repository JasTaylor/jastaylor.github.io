Bon Voyage!

Completed in 4 days, this application allows a user to find new places to visit around the world. A user can create new posts with pictures and filter places by Country. I built the backend using MongoDB, Mongoose, Express, and Amazon Web Services. The front-end was built using React with Bootstrap, AWS and Javascript.

Deployed App: Bon Voyage!

BiteOut Boston

Completed in 3 days, this application is for finding new restaurants in Boston, MA. For this SPA, I built the backend using Ruby as my backend language, Rails as my framework, and PostgreSQL for my database management. I built the front end using JavaScript, HTML, CSS with Bootstrap, and Handlebars.

Deployed App: BiteOut Boston

Tic Tac Toe

I built this project so that a user can play a game of Tic Tac Toe. This is a front end SPA I built using javascript with AJAX and jQuery, HTML, CSS, and Bootstrap. The application can track how many time a player has played a game of tic-tac-toe.

Deployed App: Tic-Tac-Toe
